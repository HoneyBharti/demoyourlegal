# **App Name**: LegalEase AI

## Core Features:

- Automated Chat Assistant: AI-powered assistant to help answer basic legal questions. The AI uses a tool to incorporate knowledge of legal precedence, but will not offer legal advice.
- Corporate Information Display: Display corporate information including formation state, Registered Agent, and EIN.
- Compliance Dates Display: Show upcoming and completed compliance dates to ensure that deadlines are never missed.
- Company Documents Upload and Storage: Allow the user to upload company documents for safe storage in the cloud. Use Google Cloud Storage to hold the company data
- Banking Integration: Connect to a Mercury bank account and track expenses.
- Bookeeping Overview Display: Financial reports that display transactions, invoices, chart of accounts, and other reports

## Style Guidelines:

- Primary color: Deep blue (#2E3192), evoking trust and professionalism.
- Background color: Light gray (#F4F4F8), providing a clean and modern backdrop.
- Accent color: Teal (#2ECC71), offering a pop of color that symbolizes growth and prosperity.
- Body and headline font: 'Inter' for a clean, modern, and readable experience. This sans-serif font will offer clarity and consistency across the platform.
- Code font: 'Source Code Pro' for displaying code snippets.
- Use clear and concise icons from Lucide React to illustrate key features and navigation elements.
- Employ subtle transitions and animations to enhance user interaction, such as hover effects and loading states. Use animations sparingly to avoid distraction.
